# CNHX — Offshore RMB Stablecoin (RXW Holding, proposed)

**Status:** Pre‑application (FSRA/ADGM).  
**Chains:** Tron (phase 1), Ethereum (phase 2).  
**Peg:** 1 CNHX = 1 CNH (offshore RMB).  
**Reserves:** CNH bank deposits + short‑term government bills.  
**Transparency:** Daily EOD reconciliation; monthly attestation; periodic audits.

## Repository Map
```
/whitepaper/      — Public PDFs (technical whitepaper, model summary)
/governance/      — Roles matrix, reserve transparency policy
/disclosures/     — Monthly reserve reports (placeholders until go‑live)
/security/        — Security policy and contact
/docs/            — Additional background docs (as needed)
```

### How to Use
1. Upload the latest whitepaper (e.g., `CNHX_Whitepaper_Draft_v0_2.pdf`) to `/whitepaper/`.
2. Upload governance PDFs (e.g., `RXW_Governance_Role_Matrix_Draft.pdf`, `Reserve_Transparency_Policy.pdf`) to `/governance/`.
3. Keep `/disclosures/` as placeholders until launch (add monthly reports later).
4. Update contact emails (`compliance@rxwholding.com`, `info@rxwholding.com`) once the domain is active.
